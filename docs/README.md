## 기능목록 구현 내역

1. 입력 구현 - 사용자 숫자 구현
2. 출력 구현 - 랜덤 값 출력 구현
3. Throw 구현 - 자리수 이상 종료 구현
4. Throw 구현 - 숫자 외 입력 종료 구현

> - System.Exit() 사용하면 안됨.

5. 자리수 구현
6. 자리수 제어 구현

---

## 출력 요구사항

> - 같은 자리에 같은 숫자가 올경우 스트라이크
> - 다른자리에 숫자가 있을 경우 볼로 표기
    >
- 하나도없는 경우 : 낫싱
>   - 3개의 숫자를 모두 맞힐 경우 : 3스트라이크, 3개의 숫자를 모두 맞히셨습니다 ! 게임 종료
>   - 게임시작 문구 출력 : 숫자 야구 게임을 시작합니다.
